## Summary

- Adiciona rota `/dashboard/map` com mapa Leaflet interativo (OpenStreetMap) para identificar e ranquear locais prioritários para novos eletropostos
- 9 camadas de dados independentes, painel lateral com controles em tempo real, e score composto configurável
- Integração com IBGE, Open Charge Map, Overpass API e dados ABVE locais (CSV)

## O que foi adicionado

### Camadas de dados
- **Renda per capita por UF** — choropleth via IBGE SIDRA API
- **Frota EV por UF** — choropleth via CSV ABVE (`public/data/abve/frotas_ev_por_uf.csv`)
- **Carregadores instalados** — marcadores via Open Charge Map API (fallback mock sem chave)
- **Shoppings / hipermercados** — clusters via Overpass API, dinâmico por bbox
- **Postos de gasolina** — clusters via Overpass API (candidatos a retrofit)
- **Rodovias principais** — polylines via Overpass API
- **Gap ABVE** — choropleth frotas vs eletropostos por UF (critico → saturado)
- **Score de Oportunidade** — choropleth + heatmap com 6 pesos ajustáveis em tempo real

### Score composto
Pesos w1 (renda), w2 (frota), w3 (POIs), w4 (distancia carregador invertida), w5 (fluxo viario), w6 (gap ABVE) ajustaveis via sliders; ranking Top 10 atualiza em tempo real.

### Painel lateral
- Toggles individuais por camada
- Sliders de peso (w1–w6) com labels descritivos
- Filtro por UF
- Ranking Top 10 com classificacao Gap ABVE
- Export CSV dos locais selecionados
- Marcacao de prospectos (localStorage)

### Dados ABVE
- CSVs seed com 27 UFs em `apps/web/public/data/abve/`
- Script `npm run update-abve` (Playwright) para extracao mensal assistida
- Documentacao completa em `MAP_MODULE_README.md`
